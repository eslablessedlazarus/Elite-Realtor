# Elite Realty | Property Inquiry Portal

A minimalist, high-conversion lead capture landing page designed for boutique real estate agencies. This project combines a premium custom UI with the flexible backend power of JotForm.

## 🚀 Overview
This single-page website is engineered to provide a "luxury boutique" feel. It uses a split-screen design:
- **Left Panel:** High-impact visual branding with professional architectural photography.
- **Right Panel:** A clean, focused workspace for an embedded JotForm, removing distractions to maximize lead conversion.

## 🛠️ Installation & Setup

### 1. File Structure
Ensure your directory looks like this:
index.html # The structure of your page
styles.css # The visual styling


## 🎨 Customization

### Changing the Background Image
The luxury home image is pulled from Unsplash. To use your own property photo:
1. Open `stylie.css`.
2. Locate the `.image-panel` class.
3. Replace the URL in the `url('...')` property with your own image link or local path.


### Responsiveness
The design is fully responsive:
- **Desktop:** Side-by-side split view.
- **Tablet/Mobile:** Vertical stack (Image on top, Form on bottom) for better readability and finger-friendly inputs.
